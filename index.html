  1. HTML Links

  • Links are found in every web pages. You can click on a link and jump to another document.
  • HTML Links - Hyperlinks:- HTML links are Hyperlinks. HTML <a> tag defines a hyperlink.
  • The most important attribute of <a> element is href attribute, which holds the link's description (The URL address).
  • Links - The target Attribute:- By default, the linked page will be displayed in the current browser window. To change this type, you must specify another attribute for the link.
    The target attribute specifies where to open the linked document.
    There are many values in the target attribute but at a time we can only use only one value:
    1. <_self> :- This is the default value. The linked document will open in the same window. It is not necessary to mention the target in the link when it is a _self.
    2. <_blank> :- Opens the document in a new window or a tab.
    3. <_parent> :- Opens the document in the parent frame.
    4. <_top> :- Opens the document in the full body of the window.

  1. HTML Tables

  • HTML tables allow us to arrange data into rows and columns.
    There are several table tags which are used to make a table.
    1. <table>:- Defines a table.
    2. <th>:- Defines a header cell in a table.
    3. <tr>:- Defines a row in a table. tr stands for Table Row.
    4. <td>:- Defines a cell in a table. td stands for Table Data.
    5. <caption>:- Defines a table caption.
    6. <colgroup>:- Specifies a group of one or more columns in a table for formatting.
    7. <col>:- Specifies column properties for each column within a colgroup element.
    8. <thead>:- Group the header content in the table.
    9. <tbody>:- Groups the body content in a table.
    10. <tfoot>:- Groups the footer content in a table.

  11. <section>:- Defines a section in a document
  12. <summary>:- Defines a visible heading for a details element

  1. <HTML Forms>

  • HTML <form> element represents a document section containing some interactive controls for submitting informations. The form collects user input and sends to the server for processing.
    Form is a container that contains input elements such as checkbox,radio button, number, submit button, text, email etc.

Syntax of HTML Form:-


  • Types of HTML form elements are:-
    1. <label> :- It defines label or the name for the form element.
    2. <input> :- It is used to get input data from the form. This data can be of various types such as text, password, email, etc by changing the types.
    3. <button> :- It is a clickable button to control other elements or to execute a functionality.
    4. <select> :- It is used to create a drop-down list.
    5. <textarea> :- It is used to get long text content such as a message.
    6. <fieldset> :- It is used to draw a box around other form elements and group the related data.
    7. <legend> :- It defines caption for fieldset elements.
    8. <option> :- It is used to define options in a drop-down list.

These are the most important ones we will be studying in this section.

  • The <label> Element :- The element defines a label or name for the form element. The <label> element helps the user who have difficulty clicking on the small regions such as (checkbox or radio buttons) - because when the user clicks the text within the label element, it toggles the radio button or the checkbox.

    [IMPORTANT]:- The for attribute of the <label> should be equal to the id attribute of the <input> element to bind them together.

  • The <input> Element :- This is the most used form element. This element can be displayed in several ways, depending on the type attribute.



  • <select> :- This element defines a drop-down list.

  • <option> :- This element defines an option that can be selected. The first item in the drop down list is selected by default. To define a pre-selected option, add the selected attribute to the option.


  • <button> :- Submit button is used to submit the details of the form.


  • <textarea> :- It is used to add comments or write some reviews. It contains unlimited number of characters. The size of the text area is given by the <rows> and <cols> attribites.


  • The <fieldset> and <legend> Elements :- The <fieldset> element is used to group related data in a form.
    The <legend> element defines a caption for the <fieldset> element.



  • Radio Button in HTML Form :- We use the <input> tag to create radio buttons.
    [NOTE] :- Radio button must share the same name to be treated as a group.
    Here in radio button, the value attribute defines the unique value associated with each radio button. The value is not shown to the user, but is the value that is sent to the server on 'submit' to identify which radio button that was selected.



  • Checkbox in an HTML Form :- We use the <input> tag to create checkbox. It is a square box to tick to activate this. It used to choose more options at a time.



  • HTML <input type="range"> :- This defines a control for entering a number. This is something like a slider control. The default range is 0 To 100. However, you can set restrictions on what numbers are accepted with the attributes below.
    1. max - specifies the maximum value allowed
    2. min - specifies the minimum value allowed
    3. step - specifies the legal number intervals
    4. value - Specifies the default value

  1. FORM Validation In HTML

  • Form validation is a process where the form checks if the information provided by the user is correct or not. If the given information is incorrect the form will alert the user that they messed up and need to fix something to proceed, if not then the form will be validated and the user will be able to continue with their form registration process.

  • Basics of form validation:-
    The first thing that is important to know for form validation is the required attribute.
    The required attribute is a boolean attribute. When present, it specifies that an input field must be filled out before submitting the form.
    [NOTE]:- The required attribute works with the following input types: text, search, url, tel, email, password, date pickers, number, checkbox, radio, and file.
